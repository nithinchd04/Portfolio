Nithin Chand — Personal Portfolio
A personal portfolio website built with vanilla HTML, CSS, and JavaScript. Features a multi-page single-file architecture with smooth scroll navigation, Three.js 3D background animations, and an interactive particle orb.
🔗 Live Site
nithinchd04.github.io (update with your deployed URL)

✨ Features

Multi-page layout — Home, About, Work, Skills, and Contact sections, each filling the full viewport
Three.js background — animated 3D particle field and wireframe shapes that react to mouse movement
Interactive particle orb — canvas-based orb in the About section that responds to cursor position
Smooth scroll navigation — nav links, side dot indicators, and scroll hints for seamless page transitions
Fully responsive — works across desktop, tablet, and mobile
Custom cursor — animated cursor ring following mouse movement
Skill ticker — scrolling marquee of tech skills
No frameworks — pure HTML, CSS, and vanilla JavaScript


🛠 Built With

HTML5 & CSS3
Vanilla JavaScript (ES6+)
Three.js r128 — 3D background
Space Grotesk & Space Mono — typography
Canvas API — particle orb
